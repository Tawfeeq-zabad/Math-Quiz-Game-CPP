# 🧮 Math Quiz Console Game

A fully customizable, interactive console-based Math Quiz game built using C++. The application dynamically generates math problems based on user-selected difficulty levels and operation types, offering real-time performance feedback and comprehensive score tracking.

## ✨ Features
* Customizable Quiz Engine: Players can define the number of questions, select difficulty tiers (Easy, Medium, Hard, Mixed), and choose specific mathematical operations (Addition, Subtraction, Multiplication, Division, Mixed).
* Dynamic Visual & Audio Feedback: Features instant console background color shifts (🟢 Green for correct answers, 🔴 Red with an audio alert beep for incorrect ones). The final screen color indicates overall success (Pass/Fail).
* Advanced State Management: Efficiently tracks quiz states using an array of structures (stQuizz and stQuestion), managing individual question data, player choices, and correct answers.
* Robust Randomization: Leverages random number generation (srand, time) to dynamically craft unique arithmetic challenges tailored to the selected complexity range.
* Replayability: Includes a continuous "Play Again" loop allowing users to restart a new customized quiz instantly.

## 🛠️ Tech Stack & Concepts Used
* Language: C++
* Concepts: Enums for difficulty and operation states, Arrays of Structs for structured state tracking, Pass/Fail logic, Control Flow (Switch-case, Do-While loops).
* Platform: Windows Console API (system("color..."), system("cls")).
